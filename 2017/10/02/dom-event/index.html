事件捕获的具体流程

一个完成的事件流分三个阶段:

  1. 事件捕获阶段 (从最不具体的到具体) (window -> document -> html -> body -> ….)
  2. 处于目标阶段
  3. 事件冒泡阶段 (从最具体到最不具体)

Event 对象的常见应用

  • 阻止默认行为 event.preventDefault()
  • 阻止冒泡 event.stopPropagation()
  • 阻止事件的响应优先级 event.stopImmediateProppagation
  • event.currentTarget // 事件当前所处于的元素
  • event.target // 触发事件的元素
