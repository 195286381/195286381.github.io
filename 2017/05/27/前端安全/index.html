前端安全类

CSRF

基本概念和缩写 CSRF, 通常称为跨站请求伪造, 英文名

Cross-site request forgery 缩写 CSRF

CSRF 攻击原理

  • 登录受信网站A, 访问危险网站B.

  • 在不登出A的情况下, 访问危险网站B.

CSRF 防御措施

  • Token 验证
  • Referer 验证 (判断页面来源)
  • 隐藏令牌

XSS

XSS 跨站脚本攻击 英文名 : Cross Site Scripting

基本概念

利用合法渠道,向页面注入JS脚本.

CSRF

对比CSRF 和 XSS

  • XSS 是向你页面注入 JS 去运行, JS
  • CSRF 是利用本身的漏洞去执行相应接口, CSRF 依赖用户登录身份认证
